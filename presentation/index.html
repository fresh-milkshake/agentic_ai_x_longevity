Agentic AI x Longevity

Кейс: ИИ-агенты для извлечения биоактивности из патентов

Пайплайн для фильтрации, анализа и оценки научных публикаций и патентов

Обзор проекта

  • Автоматизация извлечения и структурирования данных о взаимодействиях лигандов и белков из патентов
  • Использование Agents Framework, OCR, OpenCV и других технологий для обработки
  • Результат: демонстрация подхода и формирование основы для последующего расширения и улучшения

Архитектура системы

  • Многоуровневая обработка: получение патентов → анализ → валидация
  • Модули: Orchestrator, Pipeline, AI-агенты, OCR, API-интеграции

Основные модули

  • Orchestrator — управление скачивнием патентов, обработкой, сохранием и остальными процессами
  • Pipeline — последовательная обработка документов и использование AI-агентов
  • AI-агенты — SearcherAgent, BioinfAgent, SupervisorAgent, FixerAgent
  • OCR и Text Extraction — извлечение текста из PDF
  • Filtering — интеграция с USPTO API и опционально с SureChEMBL

Пример: Orchestrator

class Orchestrator:
    def run(self):
        results = self._extract_texts()
        all_docs = self._collect_documents(results)
        self._process_documents(all_docs[3:])

  • Сбор документов и запуск пайплайна обработки

Источники данных

  • BindingDB — экспериментальные данные связывания
  • SureChEMBL API — патентные данные и химические структуры
  • PubChem API — биологическая активность и мишени
  • USPTO Patents — патентные документы

Контроль качества

  • Базовая валидация формата SMILES и белковых последовательностей
  • Проверка диапазонов значений активности
  • Удаление дубликатов и ручная модерация
  • Многоуровневая система валидации с ИИ-агентами
